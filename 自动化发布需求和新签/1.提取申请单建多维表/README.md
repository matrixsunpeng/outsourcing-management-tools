# 提取申请单信息至飞书多维表

从 IMS 外包管理系统自动提取申请单数据，筛选"新签"记录，读取附件内容，写入飞书多维表。

## 环境要求

- Python 3.10+
- 飞书 CLI（`@larksuite/cli`，已认证）
- Playwright 浏览器

```bash
pip install -r requirements.txt
playwright install chromium
```

## 配置

编辑 `.env` 文件设置 IMS 登录凭证：

```
IMS_USERNAME=your_account
IMS_PASSWORD=your_password
```

首次运行会自动创建飞书多维表，地址保存到 `bitable_config.json`，后续运行复用同一张表。

## 运行

```bash
python main.py
```

## 工作流程

1. **初始化多维表** — 首次创建24字段的"外包申请单"表，后续复用
2. **导出查询结果** — 登录 IMS → 设置查询条件（近7天 + 技术合作-人员类 + 审批流程结束 + CMB）→ 导出 Excel
3. **筛选新签记录** — 解析 Excel，只保留"签约性质=新签"的记录
4. **逐条提取详情** — 按合作申请单编号搜索 → 打开详情页 → 提取13个字段
5. **读取附件内容** — 下载详情页中的 Word/Excel 附件，读取全文写入"技术合作需求明细"
6. **去重写入** — 按合作申请单编号去重，新记录写入多维表

## 文件结构

```
├── main.py              # 主入口
├── config.py            # 查询条件、字段定义、日期计算
├── ims_scraper.py       # IMS 网站自动化
├── lark_bitable.py      # 飞书多维表操作
├── .env                 # 登录凭证
├── requirements.txt     # Python 依赖
└── bitable_config.json  # 多维表地址（自动生成）
```
