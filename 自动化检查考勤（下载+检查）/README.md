# 考勤合规检查工作平台

GUI 工作平台，集成了 **IMS 数据自动下载** 和 **考勤合规检查** 两大功能。

## 功能概述

### 标签页 1：下载外包数据
自动登录 IMS 系统（`https://ims.asiainfo.com`），使用 Playwright 自动化完成：

| 下载项目 | 来源页面 | 输出文件 |
|---------|---------|---------|
| 工时详细查询 | 考勤工时 → 商务经理工时查询 | `{供应商}_工时详细查询{1,2,3}-3.xlsx` |
| 在岗人员清单 | 外包报表 → 技术合作人员变化表 | `在岗人员清单.xlsx` |
| 计提报表 | 外包数据查询 → 计提 | `计提报表.xlsx` |

工时详细查询每个供应商分 3 批导出（1~10日、11~20日、21~31日）。

### 标签页 2：检查合规
对下载的数据 + 手动准备的场地签/差旅文件，执行 6 项合规检查，生成 `{月份}_{供应商}_考勤合规检查表.xlsx`。

## 依赖安装

```bash
pip install playwright pandas openpyxl
playwright install chromium
```

> **注意**：系统需安装 Google Chrome 浏览器。Playwright 使用系统安装的 Chrome（通过 `channel="chrome"` 配置），无需额外下载 Chromium。

## 使用流程

1. **启动 GUI** → 打开"下载外包数据"标签页
2. **填写参数**：
   - 登录账号 / 密码（IMS 系统账号）
   - 月份（格式如 `202604`）
   - 供应商名称（多个用逗号分隔，如 `人瑞人才,软通动力`）
   - 工作文件夹（下载目录）
3. **点击"开始下载"** → 自动登录 IMS → 下载 3 类数据 → 保存到工作文件夹
4. **切换到"检查合规"标签页**
5. **手动补充文件**：将"场地签"和"差旅"的 Excel 文件放入工作文件夹
6. **点击"检测文件"** → 确认 5 类文件均已就绪
7. **点击"确认文件完备，开始检查"** → 执行合规检查 → 查看简报 → 打开生成的 Excel

## 合规检查条件

| # | 条件 | 阈值 | 说明 |
|---|------|------|------|
| 1 | 签到合规 | >3天 | 工作日需有签到记录，请假日期不计入 |
| 2 | 签到城市匹配 | >3天 | 签到城市需与工作地一致，差旅期间豁免 |
| 3 | 工时状态合规 | >3天 | 工作日工时状态需为"项目经理审批通过" |
| 4 | 请假合规 | >3天 | 非豁免类型请假（产假/病假/婚假/陪产假豁免） |
| 5 | 计提报表 | 缺失即不合规 | 计提报表中存在有效记录（小计>0） |
| 6 | 项目编号 | 全空即不合规 | 所有工作日工时的项目编号不能全为空 |

## 常见问题

**Q: 下载时浏览器没有打开？**  
A: 当前版本浏览器默认可见。如浏览器未启动，请确认已安装 `playwright install chromium`。

**Q: 下载失败怎么办？**  
A: 查看日志区域获取详细错误信息。工作文件夹中会生成 `debug_*.png` 截图文件，帮助定位问题。

**Q: 部分文件检测不到？**  
A: 确认文件名包含对应关键字（如"场地签"、"差旅"、"在岗人员清单"、"工时详细查询"、"计提报表"）。

**Q: 合规检查结果与命令行脚本不一致？**  
A: 合规检查逻辑源自 `F:\ClaudeCode\学习检查考勤\check_compliance.py`，理论上完全一致。如遇差异请检查数据文件是否相同。
